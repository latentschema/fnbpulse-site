# fnbPulse 360 Marketing Site

A static marketing site for fnbPulse 360 (`index.html`, `style.css`, `script.js`) served locally via Vite.

## Run locally

**Prerequisites:** Node.js

1. Install dependencies: `npm install`
2. Run the dev server: `npm run dev`

## Build

`npm run build` outputs the static site to `dist/`. `npm run preview` serves that build locally.
